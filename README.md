# Simplified-C-Compiler---UWaterloo-CS241-Assignment
Compiler that converts a simplified C language called WLP4 into MIPS Assembly. Input is presented in a tokenized form using the WLP4 to WLP4i converter. https://student.cs.uwaterloo.ca/~cs241/webtools/wlp4.html

You can test the project with the provided input.wlp4i or create your own WLP4 code and run it though the above converter for a valid input with this project. The executable is provided for each step.

WLP4 Language Specifications can be found here: https://student.cs.uwaterloo.ca/~CS241/wlp4/
